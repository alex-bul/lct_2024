## Запуск

### Запуск через Docker

```commandline
docker compose up --build
```

Веб-сервис будет доступен на localhost:8000

### Запуск через Python

```commandline
pip install -r requirements.txt
cd vtb_ad_generation_lct
python manage runserver 0.0.0.0:8000
```

Веб-сервис будет доступен на localhost:8000
